# 项目概要：坦克大战游戏

## 游戏概念
一款经典俯视角坦克射击游戏，玩家控制坦克在战场中移动、射击，摧毁敌方坦克并保护己方基地。

## 核心玩法
- **移动**：上下左右移动（WASD 或 方向键）
- **射击**：空格键 或 J键，发射炮弹
- **旋转**：坦克朝向随移动方向自动调整
- **道具系统**：击毁特定敌人掉落道具，提供临时增益
- **环境破坏**：可摧毁砖墙、钢墙等障碍物
- **基地保护**：保护己方基地不被摧毁

## 目标平台
- Windows 桌面

## 美术风格
- **整体风格**：像素风格，经典坦克大战视觉风格
- **色彩**：鲜明的军绿色、黄色、灰色为主色调
- **视觉元素**：
  - 方块化的地形元素（砖墙、钢墙、草地、水域）
  - 清晰的坦克和炮弹
  - 明显的爆炸特效
  - 道具图标清晰可辨

## 关卡设计
- **环境**：网格化的封闭战场（13x13 或更大）
- **地形元素**：
  - 砖墙：可被炮弹摧毁
  - 钢墙：需要多次攻击或特殊道具才能摧毁
  - 草地：提供隐蔽，坦克进入后不可见
  - 水域：不可通过
  - 冰面：坦克移动时会有滑动效果
- **基地**：位于战场底部中央，需保护
- **视角**：固定俯视摄像机

## 技术特点
- **网格化地图系统**：使用 TileMap 构建关卡
- **坦克控制系统**：支持四方向移动和旋转
- **炮弹系统**：包含发射、碰撞检测、爆炸效果
- **敌人AI系统**：敌人坦克具有基础AI行为
- **碰撞系统**：分层碰撞设计，支持坦克、炮弹、地形、道具
- **爆炸特效系统**：使用粒子系统实现爆炸效果
- **道具系统**：包含护盾、加速、增强火力等多种道具
- **关卡管理器**：管理关卡进度、敌人生成、胜负判定
- **音效系统**：包含移动、射击、爆炸等音效

## 目标受众
全年龄段，尤其喜欢经典坦克大战和射击游戏的玩家。
